# Resident-Tracker-Reworked
Reworking and improving an old web application. This is a resident tracker that was used to track refugees and their process throughout their residence within the system in Canada.

To run - start XAMPP Apache network and in web browser go to: http://localhost:8085/Resident-Tracker-Reworked/Frontend/index.html
Start Tomcat server to be able to acces Java backend to verify go to: localhost:8082
